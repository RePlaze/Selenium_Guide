Getting Started with Selenium

Set up your development environment and write your first Selenium script

45 minutes 2 Exercises

Welcome to Your Selenium Journey!

In this first lesson, we'll set up everything you need to start automating web browsers with Selenium WebDriver. By the end of this lesson, you'll have a working environment and your first automation script running!

Learning Objectives

  • Install Java and set up your development environment
  • Create a Maven project for Selenium
  • Understand WebDriver basics
  • Write and run your first Selenium script

Step 3: Configure Maven Dependencies

Now let's add Selenium to our project. Open your pom.xml file and add these dependencies:

pom.xml 
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 
         http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>
    
    <groupId>com.seleniumhub</groupId>
    <artifactId>selenium-basics</artifactId>
    <version>1.0.0</version>
    
    <properties>
        <maven.compiler.source>11</maven.compiler.source>
        <maven.compiler.target>11</maven.compiler.target>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
    </properties>
    
    <dependencies>
        <!-- Selenium WebDriver -->
        <dependency>
            <groupId>org.seleniumhq.selenium</groupId>
            <artifactId>selenium-java</artifactId>
            <version>4.15.0</version>
        </dependency>
        
        <!-- WebDriverManager - Automatically manages driver binaries -->
        <dependency>
            <groupId>io.github.bonigarcia</groupId>
            <artifactId>webdrivermanager</artifactId>
            <version>5.6.2</version>
        </dependency>
    </dependencies>
</project>

What's WebDriverManager?

WebDriverManager automatically downloads and manages browser drivers (chromedriver, geckodriver, etc.) for you. No more manual downloads!

Step 4: Write Your First Selenium Script

Let's create a simple script that opens Google and searches for something:

FirstSeleniumScript.java 
package com.seleniumhub;

import org.openqa.selenium.WebDriver;
import org.openqa.selenium.chrome.ChromeDriver;
import io.github.bonigarcia.wdm.WebDriverManager;

public class FirstSeleniumScript {
    public static void main(String[] args) {
        // Step 1: Setup ChromeDriver using WebDriverManager
        WebDriverManager.chromedriver().setup();
        
        // Step 2: Create a new instance of ChromeDriver
        WebDriver driver = new ChromeDriver();
        
        try {
            // Step 3: Navigate to Google
            driver.get("https://www.google.com");
            
            // Step 4: Print the page title
            System.out.println("Page title is: " + driver.getTitle());
            
            // Step 5: Wait a bit to see the browser
            Thread.sleep(3000); // 3 seconds
            
        } catch (InterruptedException e) {
            e.printStackTrace();
        } finally {
            // Step 6: Close the browser
            driver.quit();
        }
    }
}

Code Breakdown:

  • WebDriverManager.chromedriver().setup() - Downloads and sets up ChromeDriver
  • new ChromeDriver() - Creates a new Chrome browser instance
  • driver.get() - Navigates to a URL
  • driver.getTitle() - Gets the page title
  • driver.quit() - Closes the browser and cleans up

Step 5: Run Your Script

To run your script:

  1. Right-click on the Java file in your IDE
  2. Select "Run 'FirstSeleniumScript.main()'"
  3. Watch as Chrome opens automatically!
  4. See the page title printed in your console

Expected Output:

Page title is: Google

Let's Add More Functionality

Now let's make our script actually search for something on Google:

GoogleSearchScript.java 
package com.seleniumhub;

import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;
import io.github.bonigarcia.wdm.WebDriverManager;

public class GoogleSearchScript {
    public static void main(String[] args) {
        // Setup
        WebDriverManager.chromedriver().setup();
        WebDriver driver = new ChromeDriver();
        
        try {
            // Navigate to Google
            driver.get("https://www.google.com");
            
            // Find the search box using its name attribute
            WebElement searchBox = driver.findElement(By.name("q"));
            
            // Type in the search box
            searchBox.sendKeys("Selenium WebDriver");
            
            // Submit the search
            searchBox.submit();
            
            // Wait to see results
            Thread.sleep(3000);
            
            // Print the new page title
            System.out.println("Results page title: " + driver.getTitle());
            
        } catch (InterruptedException e) {
            e.printStackTrace();
        } finally {
            driver.quit();
        }
    }
}

New Concepts Introduced:

  • WebElement: Represents an HTML element on the page
  • By.name(): Locator strategy to find elements by name attribute
  • sendKeys(): Types text into an input field
  • submit(): Submits a form

Troubleshooting Common Issues

Chrome opens but immediately closes

Solution: Make sure you have driver.quit() in a finally block or add a longer sleep time.

"Cannot find symbol" error

Solution: Check your imports. Make sure you've imported all necessary classes.

Maven dependencies not downloading

Solution: Right-click on project → Maven → Reload Project (or Update Project in Eclipse).
