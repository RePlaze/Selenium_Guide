Typography & Hierarchy

Typography is one of the most important elements in Apple's design language. Learn how to use fonts, sizes, and weights to create clear visual hierarchy.

What you'll learn

  • Apple's font system and when to use each weight

Apple's Font System

Apple uses the San Francisco (SF) font family across all platforms. It's designed to be highly legible at all sizes.

Implementation

/* Use system fonts for best performance */
font-family: -apple-system, BlinkMacSystemFont, 'SF Pro Display', 
             'Segoe UI', Roboto, Helvetica, Arial, sans-serif;

/* Font weights */
.ultralight { font-weight: 200; }
.light      { font-weight: 300; }
.regular    { font-weight: 400; }
.medium     { font-weight: 500; }
.semibold   { font-weight: 600; }
.bold       { font-weight: 700; }
